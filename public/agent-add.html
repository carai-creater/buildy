エージェントを追加

新しい AI エージェントを登録します。GitHub 上のコードと、ユーザーが操作する公開 UI（GitHub Pages 等の https:// URL）の両方が必須です。LLM 実行は Buildy API 経由です。詳細は docs/creator-ui.md を参照してください。

GitHub から取り込む
