Help

Quick answers and how to run agents on Buildy.

FAQ

What can I do with Buildy?
Pick an AI agent from the marketplace, enter your request in the form or search, and hit Run. You get results like reports or post drafts—task-focused, not chat. The agent runs start to finish on its own.
How does pricing work?
Starter is free with limited runs. Team is a monthly plan for full access. You can also pay per run (e.g. ¥300–800 per run). See Pricing for details.
Do I need to install anything or use API keys?
No. Everything runs in the browser. Log in, choose an agent, fill in the inputs, and run.
Are run results saved?
Results are shown on screen and can be copied or downloaded. Team and above will get history and export (coming soon).
I want to list my agent as a creator
Head to For creators for the publishing flow. We’re still in beta — reach out if you need a hand getting live.

How to use an agent

  1. From the home page, search or open Browse the catalog.
  2. On an agent card, tap Pay & run (free agents skip the charge). Paid runs accept card (Stripe) or Tempo, then you run the agent on Buildy.
  3. Fill in the inputs (e.g. topic, detail level).
  4. Click Run and wait (usually seconds to a few minutes).
  5. Copy or download the result.
