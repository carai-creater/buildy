よくある質問

Buildyでできることは？
マーケットに並んでいるAIエージェントを選び、キーワードやフォームで依頼内容を入力して「実行」するだけで、調査・レポート・投稿案などが自動で生成されます。チャットのように会話するのではなく、タスク完了まで一気に任せられるのが特徴です。
料金はどうなっていますか？
Starterプランは無料で一部エージェントをお試しできます。Teamプランは月額課金で主要エージェントをフル利用可能。各エージェントは実行ごとの従量課金（例：¥300〜800/回）も選べます。詳しくは料金プランをご覧ください。
インストールやAPIキーは必要ですか？
不要です。ブラウザだけで完結します。Buildyにログインし、使いたいエージェントを選んで入力欄に依頼を書き、実行ボタンを押すだけです。
エージェントの実行結果は保存されますか？
実行結果は画面上に表示され、その場でコピーやダウンロードが可能です。Teamプラン以上では履歴の保存・エクスポート機能を提供予定です。
クリエイターとしてエージェントを販売したい
クリエイター向けページで公開の流れをご確認のうえ、クリエイター登録からお申し込みください。

エージェントの基本的な使い方

  1. トップページでキーワードを入力するか、「一覧へ」からエージェント一覧を開く
  2. エージェント一覧の「Tempoで決済して利用」からチェックアウトへ（無料も手続き完了が必要。有料は Tempo で送金後、Buildy で実行）
  3. 表示された入力項目（例：調査トピック、粒度）に従って内容を記入する
  4. 「実行」ボタンを押し、完了まで待つ（数十秒〜数分）
  5. 表示されたレポートやテキストをコピー・ダウンロードして利用する

その他のご質問は会社情報の問い合わせ先までお願いします。
